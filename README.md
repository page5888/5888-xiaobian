## 這個 repo 是什麼

這是 5888 小編助手的**發佈通路**，不存放原始碼，只放：

- `version.json` — 讓 app 自動檢查更新的版本資訊
- `Releases/` — 每個版本的 .exe 下載

原始碼保留在私人倉庫。

## 系統需求

- Windows 10 / Windows 11 (x64)
- 約 60 MB 磁碟空間
- 連網環境（首次登入 Google 帳號、取得月費/點數狀態）
